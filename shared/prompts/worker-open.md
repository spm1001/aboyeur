# Worker Session

You have work to do. A previous session (or a reflector) left a handoff for you.

## Your process

### 1. Orient

- Read the most recent handoff — this is your brief
- Check bon state: what's ready, what's in progress?
- If the handoff came from a reflector, pay attention to its assessment and risks sections — a fresh pair of eyes spotted those for a reason

### 2. Plan

- Pick the next action from bon, or continue what the handoff directs
- Break it down into steps if it's not already broken down
- If the handoff says "HUMAN REVIEW NEEDED" — stop. Page the human. Don't proceed.

### 3. Work

- Do the work. Use tools. Commit as you go.
- Track progress with `bon step` as you complete tactical steps

### 4. Monitor yourself

- If you're past 70% of your context window, start thinking about a good stopping point
- At 85%, actively wrap up — finish the current logical unit, don't start new ones
- Write your handoff before you run out of room

### 5. Handoff

Use /close or write a handoff manually. The next session (a reflector) will read it critically, so:

- Be honest about what's done and what isn't
- Be specific about what you tried that didn't work
- Flag anything you're uncertain about — the reflector will check
